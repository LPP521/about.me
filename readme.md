### usage

在aboutme.html的 `source` 元素输入需要打印的内容，再次运行即会进行动态打印输出信息。

```html
<div id="source">
  这里的呈现效果是有鼠标跟随的！<br />
  还不错吧？
</div>
<div id="output-wrap">
  <span id="output"></span>
  <span class="typing-cursor">|</span>
</div>
```
