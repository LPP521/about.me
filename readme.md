### usage

`typing-cursor`和`typing-cursor-black`都是闪烁的黑色光标，而`typing-cursor-white`则是闪烁的白色光标。
**请注意要在`span`里面放置内容`|`**

```html
<div id="source">
  这里的呈现效果是有鼠标跟随的！<br />
  还不错吧？
</div>
<div id="output-wrap">
  <span id="output"></span>
  <span class="typing-cursor">|</span>
</div>
```
